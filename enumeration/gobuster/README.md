```markdown
# Gobuster

Gobuster is used for directory and DNS enumeration on web servers.

## Files

- `commands.txt` : contains the Gobuster commands executed  
- `1.txt`, `2.txt`, ... : outputs of the commands

## Common Flags

### Directory scanning
- `-u` : target URL  
- `-w` : wordlist file  
- `-x` : extensions to append (e.g., php, html, py)  
- `-o` : output file  

### DNS scanning
- `-d` : target domain  
- `-w` : wordlist file  
- `-o` : output file
```
